# 차진레슨

부모와 아이가 합의하는 과정을 통해 목표와 미션을 만들고, 아이가 미션을 수행하면 보상(금전, 기프티콘)을 받는 동기부여 서비스

## 주요 기능

- **목표/미션 합의**: 부모와 아이가 함께 목표를 정하고 미션을 만드는 과정
- **미션 수행 인증**: 아이가 미션 완료 후 텍스트 또는 이미지로 수행 결과를 업로드
- **댓글 커뮤니케이션**: 아이의 수행 결과에 부모가 댓글로 피드백을 주고받는 기능
- **부모 최종 컨펌**: 부모가 수행 결과를 검토하고 최종 승인하는 기능
- **보상 지급**: 미션 달성 및 승인 시 금전 또는 기프티콘 보상 지급
